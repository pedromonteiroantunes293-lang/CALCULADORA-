Calculadora em Python
Uma calculadora interativa via terminal, desenvolvida em Python. Este projeto foca em boas práticas de programação, substituindo longas cadeias de if/elif por uma estrutura mais elegante e escalável baseada em dicionários.

 Funcionalidades
Operações Básicas: Adição, Subtração, Multiplicação e Divisão.

Loop de Execução: Permite realizar múltiplos cálculos em sequência sem precisar reiniciar o programa.

Tratamento de Erros (Exceptions):

Previne a quebra do programa caso o usuário digite letras em vez de números (ValueError).

Impede erros matemáticos de divisão por zero (ZeroDivisionError).

Código Limpo: Utilização do padrão Dispatch Table (Tabela de Despacho) com a biblioteca nativa operator para mapear as ações do usuário diretamente às funções matemáticas.
